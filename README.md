# SuperfluidRepo

Superfluid を調査・学習するためのリポジトリです。

## 動かし方

- インストール

  ```bash
  yarn
  ```

- QuickStart 用のコントラクト関連

  - デプロイ

    ```bash
    yarn backend deploy --network opSepolia
    ```
